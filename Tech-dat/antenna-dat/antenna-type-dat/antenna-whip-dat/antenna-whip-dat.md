== antenna rod 

- **Type:** Whip Antenna (Monopole)

- **Features:**
  - Single element design (simpler than dipole)
  - Compact and lightweight
  - Commonly used in:
    - Receiver modules
    - Transmitters where space is limited
  - Benefits:
    - Easy to install
    - Cost-effective
  - Appearance:
    - Slim vertical rod inside heat shrink tubing
