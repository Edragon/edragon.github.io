
# M2M Interface 

## common pin definitions 

| pin        | function                       |
| ---------- | ------------------------------ |
| 3v3 or bat | direct system power supply     |
| vin        | external high voltage supply   |
| gnd        | Power Ground                   |
| p_en       | on board power chip enable pin |
| logic_pwr  | logic power level supply       |
| rx         | RXD                            |
| tx         | TXD                            |
| boot       | M2M module boot pin            |
| ..         | ..                             |
