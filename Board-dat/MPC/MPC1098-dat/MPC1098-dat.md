| name | DB9_female OUT | RPI       | note |
| ---- | -------------- | --------- | ---- |
| TXD  |                | IO14/TXD0 |      |
| RXD  |                | IO15/RXD0 |      |
| OUT  | DB9 female 4   |           |      |
| IN   | DB9 female 3   |           |      |

- Infrared sender = IO17
- Infrared receiver = IO26

- programmable button A == IO20
- programmable button B == IO21
