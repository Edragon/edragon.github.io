
# DPR1097 RS-422


## Pin definition

- GND 
- A = R +
- B = R -
- Z = T-
- Y = T +



![](2023-10-10-15-49-47.png)


## Working Principle 

To run a loop test, connect 

  - A->Y
  - B->Z
 
Connect USB-TTL bridge for example [[PL2303-dat]] or more options from [[serial-dat]] TXD to RXD, RXD to TXD, 5V-5V, GND-GND.
