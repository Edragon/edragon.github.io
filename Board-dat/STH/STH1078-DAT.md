
# STH1078-DAT

- TDS - hex bytes output from 0x0000 to 0xffff
  - For example, FA012C0041, FA header, 012C first TDS value, 0041 second TDS value.
- Flow Speed - pulse count by interrupt, e.g. arduino D2 pin 
- Temperature - analog read
