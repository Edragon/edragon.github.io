

# DPR1130 DAT STM32 offline programmer 

- hold down PB0=OFF button and plug in USB cable into OFFLINE mode 
- hold down PB1=ON button and plug in USB cable into  ONLINE mode 

- Disk Size = 4MB = on board flash 

Supported devices:
- F0
- F3xx_HD
- F3xx_MD
- F4xx_HD
- F4xx_LD
- F4xx_MD
- F7x_128
- F10x_HD
- F10x_MD
- HC32F005
- HC32L136
