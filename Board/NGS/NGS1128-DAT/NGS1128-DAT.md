

# NGS1128-DAT

## Hardware 

![](36-15-17-03-04-2023.png)

Lead out all the function pins
- TXD / RXD / RTS / CTS / DCD / DTR / RI (via logic shifter)
- ADC / 1V8 / CS / SI / CLK / SO (SPI)
- DIN / DOUT / CLK / SYNC (PCM) / IO5 / USB_BOOT
- IO1 / IO3 / U3T / SDA / U2T
- IO2 / IO4 / U3R/ SCL / U2R
- MCU_BOOT / EXT_3V3 / VBAT / GND / 5V / DM / DP / USB_V
