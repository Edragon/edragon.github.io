
# NWI1119 DAT 

### Pin Definitions 


- IO0  - ADC
- IO2 / IO8 with pull-up on-board resistors for booting 
- IO4 / IO5 with pull-up on-board resistors for I2C
- IO6 - relay 1
- IO7 - relay 2

- IO09 - PROG Button 

- IO10 - on board LED, via jumper
