On Allwinner/Sunxi boards, the bootloader (often U-Boot) loads:

- Kernel image (zImage, uImage, etc.)
- Device Tree Blob (*.dtb)
- Boot script (boot.scr or boot.cmd)
- Root filesystem (from SD card/eMMC partition)
