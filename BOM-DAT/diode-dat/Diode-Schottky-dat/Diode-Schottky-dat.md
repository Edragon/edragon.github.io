

# Diode-Schottky-dat

- [[diode-dat]] - [[Diode-Schottky-dat]]

1. 直插型 (适合面包板、洞洞板测试)
直插型通常采用黑色的圆柱形封装（如 DO-41 或 DO-201AD），带有两根金属引脚，非常适合前期搭建原型。

1N5817： 1A 电流，20V 耐压。压降极低，非常适合低压微控制器供电隔离。

1N5819： 1A 电流，40V 耐压。这是 DIY 界最著名、最常用的万金油型号。 它的耐压比 1N5817 高，适用范围更广。对于你的 ESP32 隔离电路（ESP32 峰值电流约 500mA），用这颗绰绰有余。

1N5822： 3A 电流，40V 耐压。体积比 1N5819 大一圈（引脚更粗），常用于需要通过较大电流的主电源线上。

2. 贴片型 (适合自制 PCB 打样)
如果你打算在软件里画板子，贴片型号是首选，它们体积小巧，机器贴片或手工热风枪吹焊都很方便。通常采用 SMA、SMB 或 SMC 封装。

SS14： 1A 电流，40V 耐压（相当于 1N5819 的贴片版）。常用于各种开发板的 USB 供电防反流。

SS24： 2A 电流，40V 耐压。

SS34： 3A 电流，40V 耐压（相当于 1N5822 的贴片版）。这也是极其经典的一颗料，如果你的 5V 升压模块输出端需要总的防反接保护，这颗很合适。

SS54 / SS56： 5A 电流，40V / 60V 耐压。封装更大（通常是 SMC），用于承受更高功率的电路。




普通硅二极管： 血液流过它要交大约 0.7V 的过路费。如果升压模块输出 5.0V，经过普通二极管后，ESP32 只能拿到 4.3V。虽然 ESP32 开发板上有 3.3V 降压芯片，但 4.3V 的输入电压已经比较勉强了，发热也会增加。

肖特基二极管： 它的特点是过路费极低，通常只有 0.2V 到 0.3V 左右。5.0V 经过它之后，ESP32 依然能拿到约 4.7V 到 4.8V 的优质电压，系统运行会非常稳定。
