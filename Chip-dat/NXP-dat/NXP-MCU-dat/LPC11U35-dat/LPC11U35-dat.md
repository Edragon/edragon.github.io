

# LPC11U35-dat

32-bit ARM Cortex-M0 microcontroller; up to 128 kB flash; up to 12 kB SRAM and 4 kB EEPROM; USB device; USART

The LPC11U3x are an ARM Cortex-M0 based, low-cost 32-bit MCU family, designed for 8/16-bit microcontroller
applications, offering performance, low power, simple instruction set and memory addressing together with
reduced code size compared to existing 8/16-bit architectures.

The LPC11U3x operate at CPU frequencies of up to 50 MHz
