
# RPI-MCU-dat


The `RP2354` is a secure, high-performance microcontroller from Raspberry Pi that is identical to the RP2350 but features an additional 2MB of QSPI NOR flash memory stacked directly inside the package. It is available in two pin-compatible package variants: the 30-pin RP2354A and the 48-pin RP2354B.
